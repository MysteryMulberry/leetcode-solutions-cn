# 高级解题模式

## 单调栈
适用于寻找下一个更大/更小元素
- 例题: #42接雨水, #739每日温度

## 前缀和
适用于区间求和问题
- 构建: prefix[i] = prefix[i-1] + nums[i]
- 区间和: sum(l,r) = prefix[r+1] - prefix[l]

## 差分数组
适用于区间修改问题
- 修改区间[l,r]+val: diff[l]+=val, diff[r+1]-=val

## 并查集
适用于连通性问题
- 路径压缩+按秩合并: 近O(1)

## 线段树
适用于区间查询+区间修改
- 建树: O(n), 查询/修改: O(logn)

## 字典树(Trie)
适用于字符串前缀匹配
- 插入/查找: O(字符串长度)
