# LeetCode解题模式总结

## 双指针
适用于数组/链表两端遍历，减少嵌套循环
- 例题: #11盛最多水, #42接雨水, #283移动零

## 滑动窗口
适用于连续子数组/子串问题
- 例题: #3无重复字符最长子串

## 二分查找
适用于有序数组搜索、答案单调性
- 例题: #33搜索旋转排序数组, #300最长递增子序列

## 回溯
适用于组合、排列、子集等枚举问题
- 例题: #39组合总和, #46全排列

## 动态规划
适用于最优子结构和重叠子问题
- 例题: #70爬楼梯, #118杨辉三角, #338比特位计数

## BFS/DFS
适用于树/图遍历、连通性问题
- 例题: #200岛屿数量, #226翻转二叉树

## 位运算
适用于位操作、状态压缩
- 例题: #136只出现一次的数字
