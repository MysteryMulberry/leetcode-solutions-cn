## 解题方法分类

### 哈希表
适用于需要快速查找的场景，时间复杂度O(1)

### 双指针
适用于有序数组或链表问题，减少不必要的计算

### 动态规划
适用于最优子结构问题，通过状态转移求解

### 栈
适用于匹配、嵌套结构问题
